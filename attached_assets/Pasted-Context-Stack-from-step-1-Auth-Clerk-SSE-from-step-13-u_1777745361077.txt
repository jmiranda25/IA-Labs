Context: Stack from step 1. Auth = Clerk. SSE from step 13 used for
live-updating activity counters. Everything inside Replit + Agent 4.

Goals:

DB (Drizzle):
- forum_categories: id, name, slug (unique), description, color (text
  e.g. 'blue'), order_index (int)
- forum_threads: id, category_id (fk), author_id (text fk → users.id),
  title, slug (unique within category), body (markdown), pinned (bool
  default false), locked (bool default false), created_at,
  last_activity_at (timestamptz, indexed desc)
- forum_posts: id, thread_id (fk on delete cascade), author_id (fk →
  users.id), body (markdown), parent_post_id (uuid nullable, fk →
  forum_posts.id, for nested replies), created_at, edited_at nullable
- forum_reactions: id, post_id (fk on delete cascade), user_id (fk),
  emoji (text). Unique (post_id, user_id, emoji).
- Indexes on (category_id, last_activity_at desc, pinned).

Backend (Express + Orval):
- GET    /api/forum/categories
- GET    /api/forum/categories/:slug/threads (cursor pagination,
          pinned first then last_activity_at desc)
- GET    /api/forum/threads/:id (thread + posts list with author +
          reaction summary + my-reactions)
- POST   /api/forum/threads { category_id, title, body }
- POST   /api/forum/threads/:id/posts { body, parent_post_id? }
- PATCH  /api/forum/threads/:id   (own thread within 15min, OR admin)
- PATCH  /api/forum/posts/:id     (own post within 15min, OR admin)
- DELETE /api/forum/threads/:id   (author or admin)
- DELETE /api/forum/posts/:id     (author or admin)
- POST   /api/forum/posts/:id/reactions { emoji }   (toggle)
- POST   /api/admin/forum/threads/:id/pin    → admin only
- POST   /api/admin/forum/threads/:id/lock   → admin only
- Editing window enforced server-side: reject edits after 15 minutes
  with 403 unless requester isAdmin(clerkId).
- Update last_activity_at on every new post.
- After a new post: call notify() (step 13) to alert the thread author
  (skip if the author is replying to themselves).

Frontend (artifacts/react-vite):
- /foro: list of categories with thread + post counts; color chips.
- /foro/[categoria]: thread list, pinned threads on top with 📌, others
  by last_activity_at desc. "Nuevo tema" CTA opens a shadcn <Dialog>
  with title + category select + markdown editor.
- /foro/[categoria]/[hilo]: thread body, then flat post list with
  optional indentation for parent_post_id chain (visual depth max 2).
  Reaction picker (emoji popover) on each post. Reply composer at the
  bottom; if thread.locked, show a disabled banner "Hilo cerrado".
- Edit/delete buttons visible only when (own && within 15min) || isAdmin.
- Admin-only "Pin" / "Lock" toggles in a thread actions menu.

Deliverables: complete forum experience.
